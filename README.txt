Deskripsi umum perangkat lunak
   * Aplikasi ini merupakan aplikasi edukasi geografi.
   * Aplikasi ini dibuat untuk dapat membantu anak-anak menghafalkan letak geografis suatu negara.
   * Aplikasi ini dilengkapi dengan gambar bendera negara dan ciri khas dari negara tersebut saat pertanyaan yang diajukan dijawab dengan benar.
   * Hal ini untuk membantu anak lebih senang dan tertarik mempelajari Kewilayahan Negara di Dunia untuk anak Sekolah Dasar (SD).

Panduan Instalasi
   * Aplikasi GeografiAsyik merupakan aplikasi untuk Android yang dapat dijalankan pada Android minimum versi2.2 (Froyo)
   * Download Aplikasi berekstensi .apk (GeografiAsyik.apk)
   * Transfer ke device Android Anda dengan menggunakan Bluetooth atau dengan menggunakan kabel data
   * Cari ke direktori dimana anda menyimpan aplikasi tersebut dan double click untuk menginstall
   * Akan muncul pertanyaan "Apakah anda ingin menginstall aplikasi ini?" kemudian pilih install
   * Setelah selesai akan ada pemberitahuan apakah kita ingin membuka aplikasi tersebut atau tidak?
   * Pilih open untuk membuka aplikasi.
